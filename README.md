# Product Listing Page with Filtering & Sorting

A responsive Product Listing Page built using HTML, CSS, and JavaScript. Users can browse products, filter them by category, and sort them by price.

## 🚀 Features

- Display 18 products dynamically
- Filter products by category
- Sort products by price:
  - Low to High
  - High to Low
- Responsive Grid Layout
- Modern Product Cards
- Mobile-Friendly Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
